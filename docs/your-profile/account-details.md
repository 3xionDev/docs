# Account details

### Details

Here you can set your personal information:

- Name
- Username
- Email (can be set to not be shared publicly)
- Bio
- Company
- Job Title
- Twitter Account
- GitHub Account
- Hashnode account
- Website
- Subscribe to the Weekly recap: ON/OFF
- FAQ
- Logout

Here you can see the details of your Account for [daily.dev](https://daily.dev)
